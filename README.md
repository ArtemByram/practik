# Автосервис Про 🚗

Система автоматизации для автосервиса с удобным интерфейсом на PyQt5.

## Возможности

- 📦 Учет клиентов, автомобилей, сотрудников
- 🔧 Справочники услуг и запчастей
- 📝 Создание и управление заказами-нарядами
- 📊 Отчеты по выручке и услугам
- 🏭 Складской учет с контролем остатков
- 👥 Ролевая модель (админ/менеджер)

## Технологии

- Python 3.8+
- PyQt5
- SQLite3

## Установка

```bash
# Клонировать репозиторий
git clone https://github.com/yourusername/autoservice-pro.git

# Перейти в папку
cd autoservice-pro

# Запустить
python main.py
