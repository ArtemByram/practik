# Информационная система автосервиса "АВТОСЕРВИС"

Данный проект представляет собой информационную систему, разработанную на языке программирования Python с использованием PyQt5, предназначенную для автоматизации работы автосервиса. Система позволяет удобно управлять базой данных клиентов, автомобилей, сотрудников, услуг, запчастей и заказов-нарядов.

## Функционал системы

- **Авторизация и разграничение прав доступа** (администратор/менеджер) с защитой паролей
- **Ведение справочников:**
  - Клиенты (ФИО, телефон, email, адрес)
  - Автомобили (марка, модель, госномер, VIN, владелец)
  - Сотрудники (ФИО, должность, контакты)
  - Услуги (код, название, цена, категория)
  - Запчасти (код, название, цена, количество на складе)
- **Создание и управление заказами-нарядами** с возможностью добавления услуг и запчастей
- **Автоматический расчет итоговой стоимости заказа**
- **Складской учет** с контролем минимальных остатков
- **Движение товаров** (приход/расход запчастей)
- **Формирование отчетов:**
  - Отчет по выручке
  - Отчет по популярности услуг
- **Поиск по всем справочникам**
- **Смена пароля и восстановление доступа**
- **Блокировка после неудачных попыток входа**

## Требования к системе

- Python 3.8 или выше
- Библиотеки: PyQt5, sqlite3 (встроенная)
- Операционная система: Windows / Linux / macOS
